## Features

### Dual-Pane File Browser

- **Side-by-side panels** ‚Äî each pane with its own tab strip, history stack, and address bar
- **Multi-tab support** ‚Äî open, close, duplicate, and switch tabs per pane (`Ctrl+T`, `Ctrl+W`, `Ctrl+D`)
- **Active pane indicator** ‚Äî accent-colored border highlights the focused panel
- **F-key toolbar** ‚Äî bottom bar with 10 function buttons for common operations (Copy, Move, Rename, Search, Compare, etc.)
- **Cross-pane operations** ‚Äî copy (`F5`) or move (`F6`) files between panels in one keystroke
- **Drag & drop** ‚Äî drag files between panes (copy by default, `Shift` to move)
- **Rubber-band selection** ‚Äî click-and-drag to draw a selection rectangle
- **Status bar** ‚Äî item count, selection size, and load time

### Five View Modes

| Mode | Description |
|------|-------------|
| **Details** | Columns for icon, name, type, size, and modified date with sortable headers |
| **Large Icons** | 48√ó48 grid with wrapped file names |
| **Small Icons** | 16√ó16 compact horizontal grid |
| **List** | Single-column vertical list with 16√ó16 icons |
| **Tiles** | 32√ó32 icons with name, size, and type metadata |

### Navigation

- **Back / Forward / Up** with per-pane history stacks (`Alt+Left`, `Alt+Right`, `Backspace`)
- **Editable address bar** ‚Äî type a path and press Enter
- **Folder tree sidebar** ‚Äî hierarchical tree with lazy-loaded children
- **Quick Access** ‚Äî Home, Desktop, Documents, Downloads, Pictures, Music, Videos
- **Favorites** ‚Äî pin folders for fast access (persisted to JSON)
- **Recent locations** ‚Äî last 5 visited folders, persisted across sessions
- **Drive listing** ‚Äî all logical drives with native Windows icons
- **Refresh** ‚Äî `Ctrl+R` reloads directory contents

### File Operations

- **Copy / Cut / Paste** (`Ctrl+C`, `Ctrl+X`, `Ctrl+V`) with async operation engine
- **Copy as Path** (`Ctrl+Shift+C`) ‚Äî copies full paths to clipboard
- **Delete** (`Del`) ‚Äî Recycle Bin integration with confirmation dialog
- **Rename** (`F2`) ‚Äî inline rename that selects only the name (not the extension)
- **New Folder** (`F7`) / **New File** (`Ctrl+N`) ‚Äî creates and enters rename mode
- **Open With** ‚Äî Windows "Open As" dialog
- **Properties** (`Alt+Enter`) ‚Äî native Windows file properties dialog
- **Copy queue** ‚Äî sequential operations with pause, resume, and cancellation
- **Conflict resolution** ‚Äî Skip, Replace, Rename (auto-increment), or Ask
- **Progress tracking** ‚Äî bytes copied, percentage, current file name
- **Long path support** ‚Äî handles paths beyond the 260-character limit

### Integrated Terminal

Nexplorer includes a full-featured embedded terminal that rivals standalone tools.

**Terminal Panel**
- Multi-tab terminals with add, close, duplicate, and split (horizontal/vertical)
- **Shell support** ‚Äî PowerShell and CMD with configurable profiles
- **ANSI color rendering** ‚Äî full SGR support: 8 basic, 8 bright, 256 palette, and 24-bit RGB
- **Working directory sync** ‚Äî terminal tracks the active pane's directory
- **Git branch badge** ‚Äî shows branch name with ahead/behind/dirty indicators in the tab

**Smart Auto-Complete**
- **Inline ghost suggestions** (like PSReadLine) ‚Äî accept with `Tab` or `Right Arrow`
- **History completions** ‚Äî prefix + contains matching, weighted by frequency
- **File system completions** ‚Äî paths and directories
- **CLI tool suggestions** ‚Äî 16 tools including `git`, `docker`, `npm`, `dotnet`, `kubectl`, `terraform`, `cargo`, `go`, `aws`, `az`, and more
- **Bang commands** ‚Äî `!!` re-run last, `!n` run nth, `!prefix` run matching command
- **Tab completion cycling** through file system entries
- **Reverse history search** (`Ctrl+R`) ‚Äî interactive incremental search

**Built-in Commands**
- `clear` / `cls`, `alias` / `unalias`, `env` / `export`, `jobs` / `kill`, `metrics`, `theme`

**Execution Features**
- **Background jobs** ‚Äî append `&` to run commands in the background
- **Execution metrics** ‚Äî duration and exit code for every command
- **Structured error output** ‚Äî color-coded error summaries
- **Command aliases** ‚Äî user-defined, persisted, shell-specific

**Persistent Command History**
- Stored in JSON ‚Äî command text, directory, shell type, timestamp, exit code, duration
- **Filterable history panel** with text search and color-coded badges
- Up to 500 entries (configurable), survives crash and restart
- **Fuzzy search** support (configurable)

### Search Engine

- **Dedicated search window** (`F9` / `Ctrl+Shift+F`) with rich criteria:
  - Name query, root path, recursive toggle
  - **Regex support**
  - Min/max file size filter (KB)
  - Modified after/before date pickers
- **Streaming async results** via `IAsyncEnumerable` with incremental UI updates
- **Fully cancellable** ‚Äî stop a search at any time
- **Navigate from results** ‚Äî double-click to jump to the containing folder

### Batch Rename

- **Find & Replace** ‚Äî plain text or regex with case sensitivity toggle
- **Counter** ‚Äî configurable start, step, and zero-padding
- **Prefix / Suffix** ‚Äî prepend or append text
- **Case transforms** ‚Äî lowercase, UPPERCASE, TitleCase
- **Extension change** ‚Äî bulk update file extensions
- **Live preview** ‚Äî see old ‚Üí new names before committing

### Quick Edit

A built-in text editor powered by **AvalonEdit** (`F3`):

- **Syntax highlighting** ‚Äî auto-detected for 40+ file types (.cs, .js, .ts, .py, .json, .yaml, .md, .sql, and more)
- **Line numbers**, line/column indicator (Ln X, Col Y)
- **Save** (`Ctrl+S`) with encoding preservation
- **Unsaved changes indicator**
- **Encoding detection and display**
- 5 MB file size limit

### File Diff

- **Side-by-side comparison** (`Ctrl+D`) of files from left and right panes
- **LCS-based diff algorithm** with fallback for very large files
- **Color-coded lines** ‚Äî added (green), removed (red), unchanged (gray)
- **Line numbers** on each side
- **Synchronized scrolling**

**Folder Compare** ‚Äî compare entire directories between panes with status indicators: Same, Left Only, Right Only, Different.

### Git Integration

**Source Control Panel**
- Git status with staged and changed files
- Stage / Unstage individual files or all
- Discard changes / Clean untracked files
- Commit, Push, Pull
- Side-by-side diff for staged and unstaged changes
- Pending changes badge in tab header

**Git History Tab**
- Commit log with hash, author, date, and subject
- Branch listing with current branch highlighted
- Filter by branch
- Per-file history
- Changed files list per commit with diff view

**Git Branch Awareness**
- Fast branch detection via `.git/HEAD`
- Rich terminal prompt: `PS C:\repo ‚éá main ‚Üë2 *3>`
- Ahead/behind/dirty count indicators

### Preview Pane

Toggle with `F8`:

- **Images** ‚Äî .png, .jpg, .jpeg, .bmp, .gif, .ico, .webp, .tif, .tiff (with dimensions display)
- **Text / Code** ‚Äî 26+ extensions, first 4096 characters with monospaced font
- **Hex dump** ‚Äî binary files shown as hex (first 256 bytes)
- **Directory info** ‚Äî shows contained item count
- **Metadata** ‚Äî file name, type, size, modified date

### Space Radar

A built-in **disk space analyzer** with interactive treemap visualization (`Ctrl+Shift+R`). Think WinDirStat/SpaceSniffer ‚Äî right inside your file explorer.

- **Squarified treemap** ‚Äî every file and folder rendered as a proportionally-sized rectangle
- **Color-coded by file type** ‚Äî images (teal), video (salmon), audio (pink), archives (gold), code (blue), documents (red), executables (yellow), data (green)
- **Async scanning** ‚Äî live progress counter (files scanned + total bytes), never blocks the UI
- **Drill-down navigation** ‚Äî click any folder to zoom in, Backspace to go back, with breadcrumb trail
- **Hover tooltips** ‚Äî follow the mouse showing name, size, % of parent, and descendant counts
- **Keyboard-first** ‚Äî `Backspace` goes back, `F5` rescans, `Escape` closes
- **Full cancellation support** ‚Äî closing the window instantly cancels the scan
- **Theme-aware** ‚Äî follows the app's active theme using dynamic resource brushes

## Architecture

Nexplorer follows **Clean Architecture** with MVVM:

```
src/
‚îú‚îÄ‚îÄ Nexplorer.App          UI layer (WPF) ‚Äî views, converters, controls
‚îú‚îÄ‚îÄ Nexplorer.Core         Domain layer ‚Äî file enumeration, models
tests/
‚îî‚îÄ‚îÄ Nexplorer.Tests        Unit and integration tests
```

**Key engineering decisions:**
- Async streaming file enumeration via `IAsyncEnumerable`
- UI virtualization with `VirtualizingPanel` in recycling mode
- `CancellationToken` propagation throughout all async operations
- Native Windows shell icon extraction via `SHGetImageList`
- File system monitoring with `FileSystemWatcher` and debounce
- JSON-based settings with versioned schema and migration system

## Building from Source

```powershell
git clone https://github.com/Mehranh/Nexplorer.git
cd Nexplorer
dotnet build src/Nexplorer.App/Nexplorer.App.csproj
dotnet run --project src/Nexplorer.App/Nexplorer.App.csproj
```

Run tests:

```powershell
dotnet test tests/Nexplorer.Tests/Nexplorer.Tests.csproj
```
